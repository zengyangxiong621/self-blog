## 💡 核心概念

### 语言特性
- **动态类型** - 运行时确定变量类型
- **弱类型** - 自动类型转换
- **解释执行** - 无需编译直接运行
- **单线程** - 通过事件循环实现并发

### 重要概念
- **作用域和闭包** - 理解变量的生命周期
- **原型链** - JavaScript的继承机制
- **事件循环** - 异步执行的核心原理
- **this绑定** - 上下文对象的确定规则

## 🎨 代码风格

### 命名规范
```javascript
// 变量和函数：驼峰命名
const userName = 'John';
function getUserInfo() {}

// 常量：全大写下划线分隔
const API_BASE_URL = 'https://api.example.com';

// 类：帕斯卡命名
class UserManager {}
```

### 最佳实践
- 使用 `const` 和 `let`，避免 `var`
- 优先使用箭头函数
- 合理使用解构赋值
- 添加适当的注释
- 保持代码简洁易读
