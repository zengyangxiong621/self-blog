# HTML/CSS 基础教程

在这个章节中，我们将深入学习 HTML 和 CSS 的核心概念和实践应用。通过系统的学习，你将掌握构建现代网页所需的基础技能。

## 学习路径

### HTML 基础
- [HTML基础知识](/frontend/html-css/html-basics) - 学习 HTML 的基本概念、标签使用和语义化编程

### CSS 基础与进阶

## 为什么要学习 HTML/CSS？

HTML 和 CSS 是网页开发的基石：
- HTML 提供了网页的结构和内容
- CSS 控制网页的表现和样式
- 它们共同构成了现代网页的基础

通过系统学习这些基础知识，你将能够：
- 构建语义化的网页结构
- 创建美观的页面样式
- 实现响应式的用户界面
- 优化网页的可访问性

让我们开始学习吧！
